# 🔐 Password Strength Checker using Machine Learning

A Machine Learning project that predicts the strength of a password using TF-IDF vectorization and Logistic Regression.

## Features

- Password strength prediction
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression classifier
- Confusion Matrix
- Classification Report
- Interactive password testing

## Dataset

The dataset contains passwords labeled with different strength levels.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Pipeline

1. Load dataset
2. Clean data
3. TF-IDF Vectorization
4. Train-Test Split
5. Logistic Regression
6. Model Evaluation
7. Password Prediction
